# CenterText.py
This converts a text into a C comment.

Requires the input text to be in a file.  The default file looked for is 'input.txt'.

[input]
```text
Let us
center
these texts.
I
am
in the middle.
```

[console]
```bash
python CenterText.py
```

[output]
```text
    Let us
    center
 these texts.
       I
      am
in the middle.
```
